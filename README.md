# 📋 Management Systeam

Sistema de gerenciamento de tarefas via terminal desenvolvido em Java puro com JDBC e PostgreSQL, aplicando arquitetura em camadas.

## 🚀 Tecnologias

- Java 21
- JDBC
- PostgreSQL
- Lombok
- Maven

## ⚙️ Funcionalidades

- ✅ Criar tarefa
- ✅ Listar todas as tarefas
- ✅ Concluir tarefa
- ✅ Deletar tarefa
